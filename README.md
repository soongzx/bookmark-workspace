# 📑 Chrome Bookmark Workspace Manager

> 一款强大的 Chrome 浏览器扩展,以双面板工作区的方式管理和对比浏览书签,支持多列树状视图、全局搜索、四套主题切换与独立缩放控制。

## ✨ 核心特性

### 🎯 双面板对比模式
- **独立工作区**: 左右两个面板可同时展示不同的书签分支,方便对比和参考
- **灵活切换**: 鼠标悬停或点击即可激活对应面板,操作互不干扰
- **状态记忆**: 自动保存每个面板的展开路径,刷新后恢复工作状态
- **一键清空**: 快速清空面板内容,重新开始浏览

### 🗂️ 智能工作区管理
- **自动识别**: 智能检测书签栏节点,兼容不同语言环境
- **层级展开**: 横向多列树状视图,直观展示文件夹嵌套关系
- **快速导航**: 点击文件夹展开/折叠,自动管理右侧列序列
- **视觉引导**: 渐变边框宽度和透明度标识层级深度

### 🔍 高效全局搜索
- **实时过滤**: 输入即搜,200ms 防抖优化性能
- **多维匹配**: 同时搜索标题、URL 和完整路径
- **结果展示**: 最多显示 80 条匹配项,包含面包屑导航
- **快捷退出**: `Escape` 键或清除按钮快速返回浏览模式

### 🎨 四套精美主题
- **暗夜鎏金** (Dark Gold): 深邃黑底配金色点缀,沉稳奢华
- **暗夜深红** (Dark Crimson): 暗黑背景配红色强调,激情专业
- **海洋蓝调** (Ocean Blue): 深蓝底色配青色高亮,冷静科技
- **暖阳浅棕** (Warm Light): 米白底色配棕色装饰,温和护眼
- **即时切换**: 点击标题栏主题圆点,界面无缝过渡
- **持久化存储**: 主题偏好自动保存

### 🔎 独立缩放控制
- **面板独立**: 左右面板可设置不同缩放比例(0.6x ~ 1.8x)
- **精细调节**: 步长 0.1,适配不同屏幕和阅读习惯
- **影响范围**: 列宽、字体、间距同步调整
- **状态保持**: 缩放值持久化,下次打开自动应用

### 💫 流畅交互体验
- **拖放加载**: 支持从外部拖入文件夹直接加载到指定面板
- **Toast 提示**: 操作反馈即时显示,1.6 秒自动消失
- **平滑动画**: 列展开/折叠带滚动动画,视觉连贯
- **键盘友好**: `Ctrl/Cmd + F` 快速聚焦搜索框

## 🚀 快速开始

### 安装步骤

1. **获取源代码**
   ```bash
   git clone <repository-url>
   cd bookmark-workspace
   ```

2. **准备图标文件**(必需)
   
   从以下任一来源下载四个尺寸的 PNG 图标,放入 `icons/` 目录:
   - [iconfont](https://www.iconfont.cn/) - 阿里巴巴矢量图标库
   - [Flaticon](https://www.flaticon.com/) - 免费图标资源
   - 自行设计或使用开源图标集
   
   需要以下尺寸:
   - `icons/icon16.png` (16×16 像素)
   - `icons/icon32.png` (32×32 像素)
   - `icons/icon48.png` (48×48 像素)
   - `icons/icon128.png` (128×128 像素)

3. **加载到 Chrome**
   - 打开 Chrome 浏览器,访问 `chrome://extensions`
   - 开启右上角 **"开发者模式"** 开关
   - 点击左上角 **"加载已解压的扩展程序"** 按钮
   - 选择 `bookmark-workspace` 文件夹根目录

4. **开始使用**
   - 点击浏览器工具栏的扩展图标 📑
   - 自动在新标签页打开书签工作区界面
   - 首次使用会自动读取您的书签数据

### 系统要求

- **浏览器**: Chrome 88+ 或其他 Chromium 内核浏览器(Edge 88+, Brave 等)
- **权限**: 
  - `bookmarks` - 读取和管理书签树
  - `storage` - 保存用户偏好设置(主题、缩放值)
- **操作系统**: Windows / macOS / Linux

## 📖 使用指南

### 基础操作

#### 工作区浏览
1. **选择工作区**: 点击顶部筛选按钮(如"全部"、"前端开发"、"学习资料")
2. **展开文件夹**: 点击任意文件夹项,在其右侧新增一列展示子内容
3. **折叠分支**: 点击已展开的文件夹,折叠其右侧所有列
4. **打开书签**: 点击书签项 → 在新标签页中打开

#### 双面板模式
1. **激活面板**: 鼠标悬停或点击左侧/右侧面板
2. **独立操作**: 每个面板可展开不同的书签分支,互不影响
3. **对比查看**: 适合对照文档、比较代码库、参考不同资源
4. **清空面板**: 点击面板标题旁的清空按钮,重置该面板

#### 搜索功能
1. **输入关键词**: 在顶部搜索框输入文字,实时显示结果
2. **查看路径**: 搜索结果包含完整的面包屑路径,便于定位
3. **快速跳转**: 点击结果项直接打开书签
4. **退出搜索**: 按 `Escape` 键或点击清除按钮返回浏览模式

### 高级功能

#### 主题切换
点击标题栏右侧的四个圆形按钮:
- 🟡 **暗夜鎏金**: 深色背景 + 金色强调色,适合夜间工作
- 🔴 **暗夜深红**: 暗黑底色 + 红色高亮,专业且醒目
- 🔵 **海洋蓝调**: 深蓝基调 + 青色点缀,科技感十足
- 🟤 **暖阳浅棕**: 浅色背景 + 棕色装饰,日间护眼首选

#### 缩放调整
每个面板右上角有独立的缩放控件:
- 点击 `−` 缩小,点击 `+` 放大
- 缩放范围: 0.6x ~ 1.8x,步长 0.1
- 左面板和右面板可设置不同缩放比例
- 缩放值自动保存,下次打开时保持

#### 拖放加载
1. 从其他应用或文件管理器拖拽文件夹到面板区域
2. 释放鼠标后,该文件夹内容会加载到对应面板
3. 适合快速导入常用书签集合

#### 缓存管理
- 点击标题栏的垃圾桶图标可清除本地缓存
- 清除后会重新从 Chrome 读取最新书签数据
- 不影响浏览器实际书签,仅清除扩展的缓存状态

#### 2. 智能书签栏识别
```javascript
// 兼容多种命名方式
function findBookmarksBar(root) {
    // 优先通过 ID 识别
    const byId = root.children.find(c => c.id === '1');
    if (byId?.children) return byId;
    
    // 其次通过标题关键词
    const byTitle = root.children.find(c => {
        const t = (c.title || '').toLowerCase();
        return t.includes('书签栏') || t.includes('bookmarks bar');
    });
    
    // 降级策略:排除特殊文件夹
    // ...
}
```

#### 3. 扁平化搜索索引
```javascript
// 初始化时构建扁平数组,避免重复遍历
let allBookmarksFlat = [];

function flattenBookmarks(node, path, result) {
    if (!node) return;
    const cur = [...path];
    if (node.title && path.length > 0) cur.push(node.title);
    if (node.url) {
        result.push({
            id: node.id,
            title: node.title,
            url: node.url,
            path: cur,
            pathStr: cur.join(' ▸ ')
        });
    }
    if (node.children) {
        node.children.forEach(c => flattenBookmarks(c, cur, result));
    }
}
```

### 性能优化

- ✅ **搜索防抖**: 200ms 延迟,减少无效计算
- ✅ **扁平化索引**: 一次性构建,搜索时仅字符串匹配
- ✅ **增量渲染**: 列切换时局部更新,避免全量重绘
- ✅ **事件委托**: 减少 DOM 监听器数量
- ✅ **CSS 硬件加速**: 使用 `transform` 和 `opacity` 实现动画
- ✅ **懒加载**: 仅在需要时展开子节点

### 字体与间距

- **字体栈**: `-apple-system, BlinkMacSystemFont, "SF Pro Display", "Segoe UI", "PingFang SC", "Microsoft YaHei"`
- **基础字号**: 12-13px,随缩放动态调整
- **圆角规范**: 
  - 卡片/面板: 12px
  - 按钮: 8px
  - 搜索框: 8px
- **阴影**: `0 1px 3px rgba(0,0,0,0.6)` 低层阴影,营造浮起感

### 响应式设计

- 面板采用 Flexbox 布局,自适应窗口宽度
- 列容器使用 `overflow-x: auto` 实现横向滚动
- 搜索框最小宽度限制,避免极端情况下挤压变形
- 主题切换通过 CSS 变量实现,无闪烁过渡
