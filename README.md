# 收藏夹工作区管理系统 <small>Bookmark Workspace Manager</small>

一个 Chrome 浏览器扩展，将你的书签栏顶层文件夹抽象为"工作区"，提供高效、可视化的多列树状浏览体验。

## 项目背景

由于个人兴趣爱好十分广泛，十多年下来浏览器收藏夹累积十几个大类，几千个收藏地址。个人系统逐层分类这些信息，衍生了一个新问题：每次使用一个 URL 时需要逐层展开收藏夹，有些 URL 藏在第 3 级、甚至第 4 级目录下，十分不方便。尝试了很多浏览器和收藏夹管理工具，没有符合自己习惯的，最终决定把自己的想法实现出来，做了这个项目。

## 功能特点

- **工作区概念**：自动提取书签栏下的文件夹作为独立工作区，屏蔽无关层级
- **多列树状视图**：横向展开，层级清晰，支持点击文件夹进行展开/折叠
- **双面板模式**：可同时并排查看两个工作区，支持面板间拖拽加载内容
- **全局搜索**：快速搜索所有书签，结果高亮显示
- **四套主题**：暗夜黑金、烈焰暗红、深海蓝夜、暖阳浅金，一键切换，高亮色协调一致
- **缩放控制**：每个面板独立缩放，适应不同屏幕大小
- **拖拽交互**：将筛选区的工作区按钮直接拖入面板即可加载对应内容
- **状态持久化**：自动记住主题、缩放、面板路径等配置，打开即恢复
- **清理缓存**：一键重置所有状态
- **现代 UI**：简洁的设计，半透明工具栏，细小滚动条，跟随鼠标的拖拽提示

## 安装方法

1. 下载所有文件到本地文件夹（如 `bookmark-workspace`）
2. 打开 Chrome 浏览器，访问 `chrome://extensions`
3. 开启右上角的 **"开发者模式"**
4. 点击 **"加载已解压的扩展程序"**，选择刚才的文件夹
5. 点击浏览器工具栏的扩展图标，即可在新标签页使用

## 使用说明

- **默认界面**：打开后仅显示一个展示区，初始无内容，筛选区显示所有工作区按钮
- **加载内容**：
  - 点击筛选区的任一按钮（或拖拽按钮到展示区），即可加载该工作区
  - “全部”按钮可加载所有工作区的列表
- **导航**：
  - 点击展示区中的文件夹可展开下一级，再次点击折叠
  - 点击书签链接可在新标签页打开
- **面板操作**：
  - 点击面板标题栏或鼠标移入切换激活面板（高亮）
  - 每个面板右上角有缩放按钮 (A- / A+) 和清空按钮
- **双面板模式**：
  - 点击标题栏的 ⿰ 按钮可开启/关闭双面板，二者可独立操作
  - 拖动中间分隔条可调整面板宽度
- **搜索**：在顶部搜索框输入关键词，实时过滤书签，按 Esc 退出搜索
- **主题切换**：点击标题栏右侧四个圆形色块，可切换主题
- **清理缓存**：点击标题栏垃圾桶图标，可清除所有保存的状态并恢复到初始界面
