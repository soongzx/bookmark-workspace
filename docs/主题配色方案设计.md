Thought for 4s# Web 系统主题配色方案设计

## 设计理念

基于您的需求，设计了4套高端主题配色方案。每套主题都包含完整的色阶系统，支持正常、悬停、高亮、激活等多种状态。高亮效果采用透明度和光泽感设计，营造现代科技风格。

## 通用高亮/闪光效果 CSS

### 卡片悬停高亮效果

```css
.card-highlight {
  position: relative;
  transition: all 0.3s ease;
}

.card-highlight::before {
  content: '';
  position: absolute;
  inset: 0;
  border-radius: inherit;
  opacity: 0;
  background: var(--accent-glow);
  transition: opacity 0.3s ease;
}

.card-highlight:hover::before {
  opacity: 1;
}

.card-highlight:hover {
  box-shadow: 0 0 20px var(--accent-glow),
              0 0 40px var(--accent-glow);
}
```

### 边框闪光动画

```css
@keyframes border-shine {
  0% {
    background-position: -200% center;
  }
  100% {
    background-position: 200% center;
  }
}

.shine-border {
  position: relative;
  overflow: hidden;
}

.shine-border::after {
  content: '';
  position: absolute;
  inset: 0;
  padding: 1px;
  border-radius: inherit;
  background: var(--shine-gradient);
  background-size: 200% 100%;
  -webkit-mask: 
    linear-gradient(#fff 0 0) content-box,
    linear-gradient(#fff 0 0);
  mask: 
    linear-gradient(#fff 0 0) content-box,
    linear-gradient(#fff 0 0);
  -webkit-mask-composite: xor;
  mask-composite: exclude;
  animation: border-shine 3s linear infinite;
}
```

### 按钮高亮悬停

```css
.btn-glow {
  transition: all 0.3s ease;
}

.btn-glow:hover {
  box-shadow: 
    0 0 10px var(--accent),
    0 0 20px rgba(var(--accent-rgb), 0.4),
    0 0 30px rgba(var(--accent-rgb), 0.2);
  transform: translateY(-2px);
}

.btn-glow:active {
  transform: translateY(0);
  box-shadow: 0 0 10px var(--accent);
}
```

## 使用建议

1. **高亮透明度**：建议范围 `0.1 - 0.3`，过高会影响阅读
2. **闪光动画**：时长建议 `2-4秒`，过快会分散注意力
3. **光晕层级**：使用 `box-shadow` 多层叠加，营造深度感
4. **状态过渡**：所有状态变化添加 `transition: all 0.2s ease`
